# Template Dash

## Prérequis

1. Créez un environnement virtuel et activez-le :

   python -m venv env

   - **Pour activer l'environnement** :
     - Sur macOS/Linux :
       
       source env/bin/activate
     
     - Sur Windows :
       
       .\.venv\Scripts\activate

2. Installez les dépendances :

   pip install -r requirements.txt

## Exécution du Projet

   python app.py
